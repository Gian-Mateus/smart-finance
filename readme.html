Getting Started

Prerequisites

This project requires the following dependencies:

  • Programming Language: PHP
  • Package Manager: Composer, Npm

Installation

Build smart-finance from the source and install dependencies:

  1. Clone the repository:

    ❯ git clone https://github.com/Gian-Mateus/smart-finance

  2. Navigate to the project directory:

    ❯ cd smart-finance

  3. Install the dependencies:

Using composer:

❯ composer install

Using npm:

❯ npm install

Usage

Run the project with:

Using composer:

php {entrypoint}

Using npm:

npm start

Testing

Smart-finance uses the {test_framework} test framework. Run the test suite with:

Using composer:

vendor/bin/phpunit

Using npm:

npm test
